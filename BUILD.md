# 开发指南

## 一、准备

### :hammer: 开发工具
- WebStorm / VS code
- npm
- Vue Devtools

### :globe_with_meridians: 开发环境

- 自动安装（如果自动安装失败请手动安装）
```
npm install
```

- 手动安装
```
npm i element-ui -S
npm install --save vue-router
npm install --save nprogress
npm install
```
