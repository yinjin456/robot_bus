# 开发指南

在进行开发和构建前请先安装环境

- 自动安装（如果自动安装失败请手动安装）
```
npm install
```

- 手动安装
```
npm i element-ui -S
npm install --save vue-router
npm install
```

## 构建

### 热重启编译 （开发用）
```
npm run serve
```

### 压缩编译 （产品用）
```
npm run build
```

### 运行测试
```
npm run test
```

### 分析修复文件
```
npm run lint
```
