# robot_bus

机器人总线项目前端部分



## 项目设置

- 自动安装
```
npm install
```

- 手动安装
```
npm i element-ui -S
npm install
```

### 热重启编译 （开发用）
```
npm run serve
```

### 压缩编译 （产品用）
```
npm run build
```

### 运行测试
```
npm run test
```

### 分析修复文件
```
npm run lint
```
